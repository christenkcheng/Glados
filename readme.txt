To run Glados locally:
1.  Make a copy of 'run_glados_template.bat' and place in the same directory.
2.  Rename the copy to something like 'username_run_glados.bat'
3.  Set the copy to be ignored by SVN.
4.  Edit the copy based on current needs.
5.  Double-click the copy to run Glados.

To run Glados through Jenkins view documentation here:
https://confluence.internetbrands.com/display/QA/GLaDOS+Guides+and+Tutorials
